# 🎓 高考志愿专业组管理工具

> 一款纯前端、无需联网（除首次加载库外）的高考志愿数据整理助手，帮助考生轻松实现“冲、稳、保”梯度排序。

## ✨ 核心功能
- **📂 智能导入**：一键上传 Excel / CSV 文件，自动识别“学校代码”、“专业组”等关键字段，并将数据智能聚合为专业组卡片。
- **🖱️ 拖拽排序**：支持对**整个专业组**及**组内单个专业**进行拖拽调整顺序，随心搭建志愿梯度。
- **📝 灵活编辑**：可修改专业的**最低/最高分**，也可为专业组或单个专业添加**个性化附注/备注**，分数区间实时更新。
- **📖 视图切换**：一键“全部展开”或“全部收起”，清晰对比不同组别间的分数层次。
- **⬇️ 一键导出**：将所有调整导出为结构完整的 Excel 文件，便于打印或存档。

## 🚀 快速上手
1. 打开工具页面，点击“选择文件”上传你的志愿表（需包含：学校代码、学校名称、专业组、专业等列）。
2. 在卡片视图中，**拖拽**调整专业组或专业的优先级顺序。
3. 点击专业旁的 **✎** 按钮修改预估分数，点击 **📝** 添加备注提醒。
4. 调整完毕后，点击 **⬇️ 导出** 保存新表格。

## 📋 数据模板要求（重要）
请确保你的 Excel/CSV 文件包含以下任意列名（工具自动匹配）：
- **学校代码** / **学校名称** / **专业组** / **专业** / **最低分** / **最高分** / **备注**

*（若无表头，请按此顺序排列数据列）*
